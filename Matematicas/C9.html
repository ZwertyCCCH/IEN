Sistema de Ecuaciones 2×2

02 / 20

Un sistema de ecuaciones 2×2 tiene 2 ecuaciones con 2 incógnitas (\(x\) e \(y\)). La forma general es:

\[ \begin{cases} ax + by = c \\ dx + ey = f \end{cases} \] a, b, c, d, e, f ∈ ℝ
📌 Ejemplo 1
\(\begin{cases} x + y = 2 \\ 2x - y = 5 \end{cases}\)

\(a=1, b=1, c=2\)
\(d=2, e=-1, f=5\)
📌 Ejemplo 2
\(\begin{cases} -3x - 5y = 9 \\ 6x + 10y = -18 \end{cases}\)

\(a=-3, b=-5, c=9\)
\(d=6, e=10, f=-18\)
📌
Solución: Es un par ordenado \((x,y)\) que satisface ambas ecuaciones simultáneamente.

Tipos de Soluciones

03 / 20

Un sistema de ecuaciones 2×2 puede tener 3 tipos de soluciones, según la relación entre las rectas que representan.

✅ Solución única
¿Cuándo ocurre?
Las rectas se intersectan en un solo punto.
Condición: \(\frac{a}{d} \neq \frac{b}{e}\)
\(\begin{cases} x + y = 3 \\ x - y = 1 \end{cases}\)
Solución: \((2, 1)\)
❌ Sin solución
¿Cuándo ocurre?
Las rectas son paralelas (nunca se intersectan).
Condición: \(\frac{a}{d} = \frac{b}{e} \neq \frac{c}{f}\)
\(\begin{cases} x + y = 3 \\ x + y = 2 \end{cases}\)
Solución: \(\varnothing\)
♾️ Infinitas soluciones
¿Cuándo ocurre?
Las rectas son coincidentes (son la misma recta).
Condición: \(\frac{a}{d} = \frac{b}{e} = \frac{c}{f}\)
\(\begin{cases} x + y = 3 \\ 2x + 2y = 6 \end{cases}\)
Solución: \((x, 3-x)\)
📌
Regla práctica: Compara las razones \(\frac{a}{d}\), \(\frac{b}{e}\) y \(\frac{c}{f}\) para determinar el tipo de solución.

Método de Sustitución

04 / 20

El método de sustitución consiste en despejar una variable en una ecuación y sustituir en la otra.

📌 Pasos
  1. Despejar una variable en una de las ecuaciones.
  2. Sustituir esa expresión en la otra ecuación.
  3. Resolver la ecuación resultante.
  4. Calcular el valor de la otra variable.
  5. Verificar en ambas ecuaciones.
📌 Ejemplo
\(\begin{cases} x + y = 5 \\ 2x - y = 4 \end{cases}\)

Paso 1: Despejo \(x\) en la primera: \(x = 5 - y\)
Paso 2: Sustituyo en la segunda: \(2(5 - y) - y = 4\)
Paso 3: Resuelvo: \(10 - 2y - y = 4 \rightarrow -3y = -6 \rightarrow y = 2\)
Paso 4: Calculo \(x\): \(x = 5 - 2 = 3\)
Verificación: \(3 + 2 = 5\) ✅ · \(2(3) - 2 = 4\) ✅
Solución: \((3, 2)\)
MÉTODOS

Método de Igualación

05 / 20

El método de igualación consiste en despejar la misma variable en ambas ecuaciones y igualar las expresiones.

📌 Pasos
  1. Despejar la misma variable en ambas ecuaciones.
  2. Igualar las expresiones obtenidas.
  3. Resolver la ecuación resultante.
  4. Calcular el valor de la otra variable.
  5. Verificar en ambas ecuaciones.
📌 Ejemplo
\(\begin{cases} x + y = 6 \\ 2x - y = 3 \end{cases}\)

Paso 1: Despejo \(y\) en ambas:
\(y = 6 - x\) y \(y = 2x - 3\)
Paso 2: Igualo: \(6 - x = 2x - 3\)
Paso 3: Resuelvo: \(-x - 2x = -3 - 6 \rightarrow -3x = -9 \rightarrow x = 3\)
Paso 4: Calculo \(y\): \(y = 6 - 3 = 3\)
Verificación: \(3 + 3 = 6\) ✅ · \(2(3) - 3 = 3\) ✅
Solución: \((3, 3)\)

Método de Reducción

06 / 20

El método de reducción consiste en sumar o restar las ecuaciones para eliminar una variable.

📌 Pasos
  1. Multiplicar (si es necesario) las ecuaciones para que una variable tenga coeficientes opuestos.
  2. Sumar las ecuaciones para eliminar esa variable.
  3. Resolver la ecuación resultante.
  4. Calcular el valor de la otra variable.
  5. Verificar en ambas ecuaciones.
📌 Ejemplo
\(\begin{cases} 3x + 2y = 12 \\ 5x - 2y = 8 \end{cases}\)

Paso 1: Los coeficientes de \(y\) son \(2\) y \(-2\) (opuestos).
Paso 2: Sumo las ecuaciones:
\(3x + 2y = 12\)
\(5x - 2y = 8\)
\(8x = 20\)
Paso 3: \(x = \frac{20}{8} = 2.5\)
Paso 4: Calculo \(y\): \(3(2.5) + 2y = 12 \rightarrow 7.5 + 2y = 12 \rightarrow y = 2.25\)
Verificación: \(5(2.5) - 2(2.25) = 12.5 - 4.5 = 8\) ✅
Solución: \((2.5, 2.25)\)
MÉTODOS

Resumen de Métodos

07 / 20

Cada método tiene sus ventajas según el sistema. Elige el más adecuado para cada caso.

📌 Sustitución
Ventaja: Simple cuando una variable ya está despejada.
Cuándo usar: Cuando una ecuación tiene coeficiente 1 en \(x\) o \(y\).
Ejemplo: \(y = 2x + 1\)
📌 Igualación
Ventaja: Útil cuando ambas ecuaciones están despejadas para la misma variable.
Cuándo usar: Cuando \(y = ...\) en ambas ecuaciones.
Ejemplo: \(y = 3x - 2\) y \(y = -x + 6\)
📌 Reducción
Ventaja: Rápido cuando hay coeficientes opuestos.
Cuándo usar: Cuando al sumar/restar se elimina una variable fácilmente.
Ejemplo: \(2x + y = 5\) y \(3x - y = 10\)
💡
Consejo: Practica los tres métodos para identificar cuál es más eficiente en cada sistema.

Ejemplos tipo examen

08 / 20

Ejercicios similares a las preguntas PAES sobre sistemas de ecuaciones.

Ejemplo 1 — Sustitución
Resuelve: \(\begin{cases} x + y = 7 \\ 2x - y = 5 \end{cases}\)
Solución: \(x = 7 - y\) → \(2(7 - y) - y = 5\) → \(14 - 2y - y = 5\) → \(y = 3\) → \(x = 4\)
Respuesta: \((4, 3)\)
Ejemplo 2 — Igualación
Resuelve: \(\begin{cases} y = 2x + 3 \\ y = -x + 6 \end{cases}\)
Solución: \(2x + 3 = -x + 6\) → \(3x = 3\) → \(x = 1\) → \(y = 5\)
Respuesta: \((1, 5)\)
Ejemplo 3 — Reducción
Resuelve: \(\begin{cases} 2x + y = 8 \\ 3x - y = 7 \end{cases}\)
Solución: Sumando: \(5x = 15\) → \(x = 3\) → \(2(3) + y = 8\) → \(y = 2\)
Respuesta: \((3, 2)\)

Resolución de Problemas

09 / 20
📝 Problema 1 — Edades
La suma de las edades de dos hermanos es 30 años. La diferencia de sus edades es 6 años. ¿Cuántos años tiene cada uno?

Solución: \(x + y = 30\), \(x - y = 6\)
Sumando: \(2x = 36 \rightarrow x = 18\), \(y = 12\)
Respuesta: 18 y 12 años.
📝 Problema 2 — Precios
En una tienda, 2 lápices y 3 cuadernos cuestan $3.500. 3 lápices y 2 cuadernos cuestan $3.000. ¿Cuánto cuesta cada uno?

Solución: \(2x + 3y = 3500\), \(3x + 2y = 3000\)
Multiplicando y restando: \(6x + 9y = 10500\), \(6x + 4y = 6000\) → \(5y = 4500\) → \(y = 900\), \(x = 400\)
Respuesta: Lápiz: $400, Cuaderno: $900.
📝 Problema 3 — Números
Un número es el doble de otro. La suma de ambos números es 45. ¿Cuáles son los números?

Solución: \(x = 2y\), \(x + y = 45\) → \(2y + y = 45\) → \(y = 15\), \(x = 30\)
Respuesta: 15 y 30.

APLICACIONES

Sistemas en Contextos Navales

14 / 20

Los sistemas de ecuaciones son esenciales para modelar problemas de navegación, logística y planificación.

⚓ Velocidad y corriente
Un barco recorre 120 km río abajo en 4 horas y río arriba en 6 horas. ¿Cuál es la velocidad del barco y la corriente?

Solución: \(v+c = 30\), \(v-c=20\) → \(2v=50 \rightarrow v=25\), \(c=5\)
Respuesta: Barco: 25 km/h, Corriente: 5 km/h.
⛽ Combustible y distancia
Un buque consume 1.000 L diarios y 1.200 L diarios dependiendo de la velocidad. Si en un viaje de 15 días consumió 16.000 L, ¿cuántos días navegó a cada velocidad?

Solución: \(x+y=15\), \(1000x+1200y=16000\) → \(y=5\), \(x=10\)
Respuesta: 10 días a 1.000 L, 5 días a 1.200 L.
🧭 Tripulación
En un buque hay 120 tripulantes entre oficiales y marineros. Cada oficial tiene 2 marineros a su cargo. ¿Cuántos oficiales y marineros hay?

Solución: \(x+y=120\), \(y=2x\) → \(3x=120 \rightarrow x=40\) oficiales, \(y=80\) marineros
Respuesta: 40 oficiales, 80 marineros.

Más Aplicaciones Prácticas

15 / 20
📐 Dimensiones
El perímetro de un rectángulo es 30 m. El largo es 3 m mayor que el ancho. ¿Cuáles son las dimensiones?

Solución: \(2(L+A)=30\) → \(L+A=15\), \(L=A+3\) → \(2A+3=15\) → \(A=6\), \(L=9\)
Respuesta: Ancho: 6 m, Largo: 9 m.
📊 Inversión
Se invierten $500.000 en dos cuentas. Una da 5% y otra 8%. Si el interés anual es $31.000, ¿cuánto se invirtió en cada una?

Solución: \(x+y=500000\), \(0.05x+0.08y=31000\) → \(x=300000\), \(y=200000\)
Respuesta: $300.000 al 5%, $200.000 al 8%.
🧪 Mezcla
Se mezclan 20 L de solución al 30% con \(x\) L de solución al 50% para obtener una mezcla al 40%. ¿Cuántos litros de la segunda solución se necesitan?

Solución: \(0.3(20)+0.5x=0.4(20+x)\) → \(6+0.5x=8+0.4x\) → \(0.1x=2\) → \(x=20\) L
Respuesta: 20 L.

Ejercicios Adicionales

Ejercicio 1 — Sistema con fracciones
Resuelve: \(\begin{cases} \frac{x}{2} + \frac{y}{3} = 4 \\ \frac{x}{3} - \frac{y}{2} = 1 \end{cases}\)
Solución: Multiplicar por MCM(2,3)=6 → \(3x+2y=24\), \(2x-3y=6\) → Resolviendo: \(x=6, y=3\)
Respuesta: \((6, 3)\)

\(\begin{cases} 2x + 3y = 8 \\ 4x + 6y = 15 \end{cases}\)
Solución: \(4x+6y = 2(2x+3y) = 16 \neq 15\) → Sin solución
Respuesta: \(\varnothing\)
Ejercicio 3 — Infinitas soluciones
\(\begin{cases} x - 2y = 3 \\ 2x - 4y = 6 \end{cases}\)
Solución: \(2x-4y = 2(x-2y) = 6\) → misma ecuación → infinitas soluciones
Respuesta: \((3+2y, y)\) con \(y \in \mathbb{R}\)

Ejercicios Avanzados

Ejercicio 4 — Sistema 3×3 (M2)
\(\begin{cases} x + y + z = 6 \\ x - y + z = 4 \\ 2x + y - z = 5 \end{cases}\)
Solución: Sumando las dos primeras: \(2x+2z=10 \rightarrow x+z=5\)
De la primera: \(y=6-x-z=6-5=1\) → Resolviendo: \(x=3, z=2\)
Respuesta: \((3, 1, 2)\)
Ejercicio 5 — Problema de edades
La edad de un padre es el triple de la edad de su hijo. Hace 5 años, la suma de sus edades era 50 años. ¿Cuántos años tienen?

Solución: \(P = 3H\), \((P-5)+(H-5)=50\) → \(P+H=60\) → \(4H=60\) → \(H=15\), \(P=45\)
Respuesta: Padre: 45 años, Hijo: 15 años.
Ejercicio 6 — Sistema con parámetros
\(\begin{cases} 2x + ky = 4 \\ 3x - y = 6 \end{cases}\)
Encuentra \(k\) para que el sistema tenga solución única.
Solución: \(\frac{2}{3} \neq \frac{k}{-1} \rightarrow k \neq -\frac{2}{3}\)
Respuesta: \(k \neq -\frac{2}{3}\)

Tipos de Soluciones — Resumen Gráfico

19 / 20

Un sistema de ecuaciones 2×2 puede tener 3 tipos de soluciones según la relación entre las rectas que representan.

✅ Solución única
Rectas que se intersectan
\(\frac{a}{d} \neq \frac{b}{e}\)
\(\begin{cases} x + y = 3 \\ x - y = 1 \end{cases}\) → \((2,1)\)
❌ Sin solución
Rectas paralelas
\(\frac{a}{d} = \frac{b}{e} \neq \frac{c}{f}\)
\(\begin{cases} x + y = 3 \\ x + y = 2 \end{cases}\) → \(\varnothing\)
♾️ Infinitas soluciones
Rectas coincidentes
\(\frac{a}{d} = \frac{b}{e} = \frac{c}{f}\)
\(\begin{cases} x + y = 3 \\ 2x + 2y = 6 \end{cases}\) → \((x, 3-x)\)
📌 Resumen de condiciones
Solución única
\(\frac{a}{d} \neq \frac{b}{e}\)
Rectas se intersectan
Sin solución
\(\frac{a}{d} = \frac{b}{e} \neq \frac{c}{f}\)
Rectas paralelas
Infinitas soluciones
\(\frac{a}{d} = \frac{b}{e} = \frac{c}{f}\)
Rectas coincidentes
💡
Consejo: Para determinar el tipo de solución, compara las razones \(\frac{a}{d}\), \(\frac{b}{e}\) y \(\frac{c}{f}\). Si son todas iguales → infinitas soluciones. Si solo las dos primeras son iguales → sin solución. Si son todas distintas → solución única.

Resumen: Sistema de Ecuaciones Lineales

20 / 20
📌 Contenidos clave
  • Sistema 2×2: \(ax + by = c\), \(dx + ey = f\)
  • Solución: Par ordenado \((x,y)\) que satisface ambas ecuaciones
  • Sustitución: Despejar y reemplazar
  • Igualación: Despejar la misma variable e igualar
  • Reducción: Sumar/restar para eliminar una variable
  • Solución única: \(\frac{a}{d} \neq \frac{b}{e}\)
  • Sin solución: \(\frac{a}{d} = \frac{b}{e} \neq \frac{c}{f}\)
  • Infinitas soluciones: \(\frac{a}{d} = \frac{b}{e} = \frac{c}{f}\)
🎯 Estrategias para el examen
  • Identifica qué método es más eficiente.
  • Con coeficientes 1: sustitución o igualación.
  • Con coeficientes opuestos: reducción.
  • Verifica la solución en ambas ecuaciones.
  • En problemas verbales: traduce a sistema de ecuaciones.
  • Recuerda: \(\frac{a}{d}\), \(\frac{b}{e}\), \(\frac{c}{f}\) determinan el tipo de solución.
