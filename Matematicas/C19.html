Percentiles

02 / 20

El percentil K (\(P_K\)) es el valor que deja el \(K\%\) de los datos ordenados por debajo de él.

\[ P_K \text{ es el valor tal que } K\% \text{ de los datos son } \leq P_K \]
📌 Cálculo para datos no agrupados
Paso 1: Ordenar los datos.
Paso 2: Calcular la posición: \(\frac{K(n+1)}{100}\)
Paso 3: Si la posición es decimal, promediar los valores adyacentes.
Datos: 4, 0, 1, 2, 7, 4, 0, 9
Ordenados: 0, 0, 1, 2, 4, 4, 7, 9 (\(n=8\))
\(P_{40} = \frac{40(8+1)}{100} = 3.6\)
\(P_{40} = \frac{1 + 2}{2} = 1.5\)
📌 Cálculo para datos agrupados
Paso 1: Calcular \(\frac{K \cdot n}{100}\)
Paso 2: Buscar el intervalo que contiene esa posición (con frecuencia acumulada).
Paso 3: Si la posición es decimal, redondear al entero mayor.
Tabla: [2,4[ → 7, [4,6[ → 4, [6,8[ → 6, [8,10[ → 3 (\(n=20\))
\(P_{45} = \frac{45 \cdot 20}{100} = 9\)
Intervalo: \([4, 6[\)

Cuartiles

03 / 20

Los cuartiles dividen un conjunto de datos ordenados en 4 partes iguales.

\[ Q_1 = P_{25} \quad \text{(25% de los datos)} \] \[ Q_2 = P_{50} = \text{Mediana} \quad \text{(50% de los datos)} \] \[ Q_3 = P_{75} \quad \text{(75% de los datos)} \]
📌 Interpretación
  • \(Q_1\): El 25% de los datos está por debajo.
  • \(Q_2\): El 50% de los datos está por debajo (mediana).
  • \(Q_3\): El 75% de los datos está por debajo.
📌 Ejemplo
Datos: 4, 0, 1, 2, 7, 4, 0, 9
Ordenados: 0, 0, 1, 2, 4, 4, 7, 9

\(Q_1 = P_{25} = 1\)
\(Q_2 = P_{50} = \frac{2+4}{2} = 3\)
\(Q_3 = P_{75} = 4\)

Diagrama de Cajón (Box Plot)

04 / 20

El diagrama de cajón es una representación gráfica basada en 5 datos: mínimo, \(Q_1\), \(Q_2\), \(Q_3\) y máximo.

📌 Elementos
  • Mínimo: Valor más pequeño.
  • \(Q_1\): Primer cuartil.
  • \(Q_2\): Mediana.
  • \(Q_3\): Tercer cuartil.
  • Máximo: Valor más grande.
📌 Interpretación
  • La caja contiene el 50% central de los datos.
  • Los bigotes muestran los valores extremos.
  • Permite identificar simetría y sesgo.
📌
Tipos de sesgo: Caja simétrica → datos simétricos · Caja desplazada → sesgo positivo o negativo.

Simetría y Sesgo en el Diagrama de Cajón

05 / 20

El diagrama de cajón permite identificar la simetría o el sesgo de la distribución de los datos.

✅ Simétrica
Caja centrada
\(Q_2\) en el centro de la caja
📈 Sesgo Positivo
Bigote derecho más largo
Cola hacia la derecha
📉 Sesgo Negativo
Bigote izquierdo más largo
Cola hacia la izquierda
📌
Interpretación: Si \(Q_2 - Q_1 > Q_3 - Q_2\) → sesgo negativo. Si \(Q_2 - Q_1 < Q_3 - Q_2\) → sesgo positivo.

Ejemplos tipo examen

06 / 20

Ejercicios similares a las preguntas PAES sobre medidas de posición.

Ejemplo 1 — Percentil
Dados: 2, 4, 6, 8, 10, 12, 14, 16. Calcula el percentil 50.
Solución: \(P_{50} = \frac{50(8+1)}{100} = 4.5\) → \(\frac{8+10}{2} = 9\)
Respuesta: 9
Ejemplo 2 — Cuartiles
Dados: 3, 5, 7, 9, 11, 13, 15, 17. Encuentra \(Q_1\), \(Q_2\), \(Q_3\).
Solución: \(Q_1 = 6\), \(Q_2 = 10\), \(Q_3 = 14\)
Respuesta: \(Q_1 = 6\), \(Q_2 = 10\), \(Q_3 = 14\)
Ejemplo 3 — Diagrama de cajón
Un conjunto de datos tiene \(Mín = 5\), \(Q_1 = 10\), \(Q_2 = 15\), \(Q_3 = 20\), \(Máx = 30\). ¿Cuál es el rango intercuartílico (IQR)?
Solución: \(IQR = Q_3 - Q_1 = 20 - 10 = 10\)
Respuesta: 10

Resolución de Problemas

07 / 20
📝 Problema 1 — Percentil en calificaciones
Las calificaciones de 10 estudiantes son: 3, 4, 4, 5, 5, 6, 6, 7, 7, 8. ¿Cuál es el percentil 30?

Solución: \(P_{30} = \frac{30(10+1)}{100} = 3.3\) → \(\frac{4+5}{2} = 4.5\)
Respuesta: 4.5
📝 Problema 2 — Cuartiles de edades
Edades de 8 personas: 18, 20, 22, 25, 28, 30, 32, 35. Encuentra \(Q_1\), \(Q_2\), \(Q_3\).

Solución: \(Q_1 = 21\), \(Q_2 = 26.5\), \(Q_3 = 31\)
Respuesta: \(Q_1 = 21\), \(Q_2 = 26.5\), \(Q_3 = 31\)
📝 Problema 3 — Interpretación de diagrama
Un diagrama de cajón muestra: \(Mín = 10\), \(Q_1 = 20\), \(Q_2 = 30\), \(Q_3 = 40\), \(Máx = 60\). ¿Qué puedes concluir sobre la simetría?

Solución: \(Q_2 - Q_1 = 10\), \(Q_3 - Q_2 = 10\) → simétrico
Respuesta: Los datos son simétricos.

Los percentiles y cuartiles son esenciales en el análisis de datos navales y logísticos.

Respuesta: \(Q_1 = 132.5\), \(Q_2 = 142.5\), \(Q_3 = 152.5\)

¿Cuál es el IQR?

Resumen: Medidas de Posición

20 / 20
📌 Contenidos clave
  • Percentil K: \(K\%\) de los datos son ≤ \(P_K\)
  • Cuartiles: \(Q_1 = P_{25}\), \(Q_2 = P_{50}\), \(Q_3 = P_{75}\)
  • Rango intercuartílico: \(IQR = Q_3 - Q_1\)
  • Diagrama de cajón: Mín, \(Q_1\), \(Q_2\), \(Q_3\), Máx
  • Sesgo positivo: Cola derecha más larga
  • Sesgo negativo: Cola izquierda más larga
  • Simetría: \(Q_2 - Q_1 = Q_3 - Q_2\)
🎯 Estrategias para el examen
  • Identifica si los datos están agrupados o no.
  • Para percentiles: usa \(\frac{K(n+1)}{100}\) (no agrupados).
  • Para percentiles agrupados: usa \(\frac{K \cdot n}{100}\) y busca el intervalo.
  • Recuerda: \(Q_2\) es la mediana.
  • El diagrama de cajón muestra la distribución de los datos.
  • Interpreta el sesgo para entender la forma de la distribución.
  • El IQR mide la dispersión del 50% central.
