Definición de Ecuación de Segundo Grado

02 / 20

Una ecuación de segundo grado o cuadrática es aquella donde el mayor exponente de la incógnita es 2. Su forma general es:

\[ ax^2 + bx + c = 0 \quad \text{con } a \neq 0 \] a, b, c ∈ ℝ · a: coeficiente cuadrático · b: coeficiente lineal · c: término independiente
📌 Ejemplos
\(x^2 + 2x + 3 = 1 \rightarrow x^2 + 2x + 2 = 0\)
\(x^2 + 5x + 9 = 5 \rightarrow x^2 + 5x + 4 = 0\)
\(2x^2 + 5x - 5 = 3 \rightarrow 2x^2 + 5x - 8 = 0\)
📌 Forma general
\(ax^2 + bx + c = 0\)

\(x^2 + x + 3 = 1\) → \(a=1, b=1, c=2\)
\(x^2 - 4 = 0\) → \(a=1, b=0, c=-4\)
📌
Recuerda: \(a \neq 0\). Si \(a = 0\), la ecuación se reduce a una de primer grado.

Método de Factorización

03 / 20

El método de factorización consiste en expresar la ecuación como el producto de dos binomios e igualar cada factor a cero.

📌 Pasos
  1. Igualar la ecuación a cero.
  2. Factorizar la expresión (trinomio cuadrado perfecto, diferencia de cuadrados, etc.).
  3. Igualar cada factor a cero.
  4. Resolver cada ecuación lineal.
  5. Verificar las soluciones.
📌 Ejemplo
\(x^2 - x - 90 = 0\)

Paso 1: Factorizar: \((x + 10)(x - 9) = 0\)
Paso 2: Igualar a cero: \(x + 10 = 0\) o \(x - 9 = 0\)
Paso 3: Resolver: \(x = -10\) o \(x = 9\)
Verificación: \((-10)^2 - (-10) - 90 = 100 + 10 - 90 = 0\) ✅
\(9^2 - 9 - 90 = 81 - 9 - 90 = 0\) ✅
Solución: \(x = -10\) y \(x = 9\)
MÉTODOS

Método de Completación de Cuadrados

04 / 20

El método de completación de cuadrados consiste en transformar la ecuación en un cuadrado de binomio.

📌 Pasos
  1. Reescribir la ecuación como \(ax^2 + bx = -c\).
  2. Dividir por \(a\) si es necesario.
  3. Completar el cuadrado: sumar \(\left(\frac{b}{2}\right)^2\) a ambos lados.
  4. Escribir como \((x - h)^2 = k\).
  5. Aplicar raíz cuadrada y despejar.
📌 Ejemplo
\(x^2 + 4x + 3 = 0\)

Paso 1: \(x^2 + 4x = -3\)
Paso 2: \(\left(\frac{4}{2}\right)^2 = 4\) → sumar a ambos lados:
\(x^2 + 4x + 4 = -3 + 4\)
Paso 3: \((x + 2)^2 = 1\)
Paso 4: \(x + 2 = \pm 1\)
\(x = -2 - 1 = -3\) o \(x = -2 + 1 = -1\)
Solución: \(x = -3\) y \(x = -1\)

Fórmula General

05 / 20

La fórmula general permite resolver cualquier ecuación de segundo grado. También llamada fórmula cuadrática.

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] a, b, c son los coeficientes de \(ax^2 + bx + c = 0\)
📌 Ejemplo
\(2x^2 + 7x + 10 = x^2\)

Paso 1: Igualar a cero: \(x^2 + 7x + 10 = 0\)
Paso 2: Identificar: \(a=1, b=7, c=10\)
Paso 3: Aplicar fórmula:
\(x = \frac{-7 \pm \sqrt{49 - 4(1)(10)}}{2(1)} = \frac{-7 \pm \sqrt{9}}{2} = \frac{-7 \pm 3}{2}\)
\(x = \frac{-7 + 3}{2} = -2\) o \(x = \frac{-7 - 3}{2} = -5\)
Solución: \(x = -5\) y \(x = -2\)
📌
Recuerda: La fórmula general funciona para todos los casos, incluso cuando las raíces no son números enteros.

Discriminante (Δ)

06 / 20

El discriminante \(\Delta = b^2 - 4ac\) determina la naturaleza de las soluciones de una ecuación cuadrática.

✅ \(\Delta > 0\)
Dos raíces reales y distintas
\(x^2 - 5x + 6 = 0\)
\(\Delta = 25 - 24 = 1 > 0\)
\(x = 2\) y \(x = 3\)
✅ \(\Delta = 0\)
Una raíz real doble
\(x^2 - 4x + 4 = 0\)
\(\Delta = 16 - 16 = 0\)
\(x = 2\) (doble)
❌ \(\Delta < 0\)
Sin raíces reales
\(x^2 + 2x + 5 = 0\)
\(\Delta = 4 - 20 = -16 < 0\)
No hay solución real
📌
Importante: El discriminante te dice cuántas y qué tipo de soluciones tiene la ecuación.

Propiedades de las Raíces

07 / 20

Si \(x_1\) y \(x_2\) son las raíces de \(ax^2 + bx + c = 0\), se cumplen las siguientes propiedades:

📌 Suma de raíces
\[ x_1 + x_2 = -\frac{b}{a} \]
\(2x^2 - 8x + 6 = 0\)
\(x_1 + x_2 = -\frac{-8}{2} = 4\)
📌 Producto de raíces
\[ x_1 \cdot x_2 = \frac{c}{a} \]
\(2x^2 - 8x + 6 = 0\)
\(x_1 \cdot x_2 = \frac{6}{2} = 3\)
📌 Verificación
\(2x^2 - 8x + 6 = 0\) → \(a=2, b=-8, c=6\)
Raíces: \(x_1 = 1\), \(x_2 = 3\)
Suma: \(1 + 3 = 4 = -\frac{-8}{2}\) ✅
Producto: \(1 \cdot 3 = 3 = \frac{6}{2}\) ✅

Ejemplos tipo examen

Ejercicios similares a las preguntas PAES sobre ecuaciones de segundo grado.

Ejemplo 1 — Factorización
Resuelve: \(x^2 + 5x + 6 = 0\)
Solución: \((x+2)(x+3)=0\) → \(x=-2\) o \(x=-3\)
Respuesta: \(x = -2\) y \(x = -3\)
Ejemplo 2 — Fórmula general
Resuelve: \(x^2 - 6x + 8 = 0\)
Solución: \(a=1, b=-6, c=8\)
\(x = \frac{6 \pm \sqrt{36 - 32}}{2} = \frac{6 \pm 2}{2}\)
\(x = 4\) o \(x = 2\)
Respuesta: \(x = 4\) y \(x = 2\)
Ejemplo 3 — Discriminante
¿Cuántas soluciones reales tiene \(2x^2 + 4x + 1 = 0\)?
Solución: \(\Delta = 16 - 4(2)(1) = 16 - 8 = 8 > 0\) → 2 soluciones reales
Respuesta: 2 soluciones reales

Resolución de Problemas

09 / 20
📝 Problema 1 — Área de un rectángulo
El área de un rectángulo es \(24\) m² y su largo es \(2\) m mayor que su ancho. ¿Cuáles son las dimensiones?

Solución: Ancho = \(x\), Largo = \(x + 2\)
\(x(x + 2) = 24 \rightarrow x^2 + 2x - 24 = 0\)
\((x + 6)(x - 4) = 0 \rightarrow x = 4\) (el otro valor es negativo)
Respuesta: Ancho 4 m, Largo 6 m.
📝 Problema 2 — Caída libre
La altura de un objeto en caída libre está dada por \(h(t) = -5t^2 + 40t\). ¿En qué tiempo la altura es 75 m?

Solución: \(-5t^2 + 40t = 75 \rightarrow -5t^2 + 40t - 75 = 0\)
Dividiendo por -5: \(t^2 - 8t + 15 = 0\)
\((t - 3)(t - 5) = 0 \rightarrow t = 3\) o \(t = 5\)
Respuesta: A los 3 y 5 segundos.
📝 Problema 3 — Números consecutivos
El producto de dos números enteros consecutivos es 56. ¿Cuáles son los números?

Solución: \(x(x+1) = 56 \rightarrow x^2 + x - 56 = 0\)
\((x + 8)(x - 7) = 0 \rightarrow x = 7\) o \(x = -8\)
Respuesta: 7 y 8, o -8 y -7.

Ecuaciones Cuadráticas en Contextos Navales

14 / 20

Las ecuaciones cuadráticas modelan situaciones con aceleración, trayectorias parabólicas y optimización.

⚓ Trayectoria de un proyectil
La altura de un proyectil está dada por \(h(t) = -5t^2 + 40t + 20\), donde \(t\) es el tiempo en segundos.

¿En qué tiempo el proyectil alcanza los 80 metros?

Solución: \(-5t^2 + 40t + 20 = 80\)
\(-5t^2 + 40t - 60 = 0 \rightarrow t^2 - 8t + 12 = 0\)
\((t - 2)(t - 6) = 0 \rightarrow t = 2\) o \(t = 6\)
Respuesta: A los 2 y 6 segundos.
📐 Diseño de compartimentos
Un compartimento rectangular tiene área de \(48\) m². Su largo es \(4\) m mayor que su ancho. ¿Cuáles son las dimensiones?

Solución: \(x(x + 4) = 48 \rightarrow x^2 + 4x - 48 = 0\)
\((x + 8)(x - 6) = 0 \rightarrow x = 6\)
Respuesta: Ancho 6 m, Largo 10 m.
🧮 Optimización de recursos
El costo de producción está dado por \(C(x) = x^2 - 20x + 100\). ¿Cuántas unidades minimizan el costo?

Solución: El mínimo está en el vértice: \(x = -\frac{b}{2a} = \frac{20}{2} = 10\)
Respuesta: 10 unidades minimizan el costo.

Más Aplicaciones Prácticas

15 / 20
📊 Área de un terreno
Un terreno rectangular tiene un perímetro de \(40\) m y un área de \(96\) m². ¿Cuáles son sus dimensiones?

Solución: \(2(L + A) = 40 \rightarrow L + A = 20\)
\(L \cdot A = 96 \rightarrow A(20 - A) = 96\)
\(20A - A^2 = 96 \rightarrow A^2 - 20A + 96 = 0\)
\((A - 8)(A - 12) = 0 \rightarrow A = 8, L = 12\)
Respuesta: 8 m × 12 m.
📈 Lanzamiento de un objeto
Un objeto se lanza verticalmente con velocidad \(v_0 = 30\) m/s. La altura está dada por \(h(t) = -5t^2 + 30t\). ¿Cuándo alcanza los 45 m?

Solución: \(-5t^2 + 30t = 45\)
\(-5t^2 + 30t - 45 = 0 \rightarrow t^2 - 6t + 9 = 0\)
\((t - 3)^2 = 0 \rightarrow t = 3\)
Respuesta: A los 3 segundos.
🧪 Mezcla de soluciones
Se tienen 10 L de una solución al 30%. ¿Cuántos litros de solución al 60% se deben agregar para obtener una solución al 40%?

Solución: \(0.3(10) + 0.6x = 0.4(10 + x)\)
\(3 + 0.6x = 4 + 0.4x \rightarrow 0.2x = 1 \rightarrow x = 5\)
Respuesta: 5 litros.

Comparación de Métodos

19 / 20

Cada método tiene sus ventajas según el tipo de ecuación. Elige el más adecuado.

📌 Factorización
Ventaja: Rápido y sencillo.
Cuándo usar: Cuando la ecuación es factorizable fácilmente.
Ejemplo: \(x^2 + 5x + 6 = 0\)
📌 Completación
Ventaja: Útil para entender la estructura.
Cuándo usar: Cuando el coeficiente \(a = 1\) y \(b\) es par.
Ejemplo: \(x^2 + 6x + 5 = 0\)
📌 Fórmula General
Ventaja: Funciona siempre.
Cuándo usar: Cuando los otros métodos no son prácticos.
Ejemplo: \(3x^2 + 7x - 2 = 0\)
📌 Resumen
  • Factorización: Más rápida, pero no siempre funciona.
  • Completación: Útil para entender el concepto.
  • Fórmula General: Funciona en todos los casos.

Resumen: Ecuaciones de Segundo Grado

20 / 20
📌 Contenidos clave
  • Forma general: \(ax^2 + bx + c = 0\)
  • Factorización: \((x + a)(x + b) = 0\)
  • Completación: \((x - h)^2 = k\)
  • Fórmula general: \(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)
  • Discriminante: \(\Delta = b^2 - 4ac\)
  • Si \(\Delta > 0\): 2 raíces reales distintas
  • Si \(\Delta = 0\): 1 raíz real doble
  • Si \(\Delta < 0\): Sin raíces reales
  • Suma raíces: \(x_1 + x_2 = -\frac{b}{a}\)
  • Producto raíces: \(x_1 \cdot x_2 = \frac{c}{a}\)
🎯 Estrategias para el examen
  • Identifica el tipo de ecuación y elige el método adecuado.
  • Si es factorizable: factoriza.
  • Si \(a = 1\) y \(b\) es par: completa cuadrados.
  • Si no se puede factorizar: usa la fórmula general.
  • Para saber el número de soluciones: calcula el discriminante.
  • En problemas verbales: traduce a ecuación cuadrática.
  • Recuerda: \((a + b)^2 \neq a^2 + b^2\).
