Traslación de Funciones

02 / 20

La traslación de funciones consiste en mover la gráfica en el plano cartesiano sin cambiar su forma.

\[ g(x) = f(x - h) + k \] h: traslación horizontal · k: traslación vertical
📌 Traslación Vertical
  • \(f(x) + k\) → k unidades arriba
  • \(f(x) - k\) → k unidades abajo
\(f(x) = x^2\)
\(f(x) + 3 = x^2 + 3\) → 3 unidades arriba
\(f(x) - 4 = x^2 - 4\) → 4 unidades abajo
📌 Traslación Horizontal
  • \(f(x - h)\) → h unidades derecha
  • \(f(x + h)\) → h unidades izquierda
\(f(x) = x^2\)
\(f(x-2) = (x-2)^2\) → 2 unidades derecha
\(f(x+3) = (x+3)^2\) → 3 unidades izquierda
📌
Regla práctica: \(f(x - h) + k\) → mueve la gráfica \(h\) a la derecha y \(k\) hacia arriba.
TEORÍA

Traslación Vertical — Ejemplos

03 / 20

Ejemplos de traslación vertical para distintos tipos de funciones.

📌 Ejemplo 1 — Función Lineal
\(f(x) = 3x + 5\)

3 unidades arriba: \(u(x) = 3x + 5 + 3 = 3x + 8\)
4 unidades abajo: \(d(x) = 3x + 5 - 4 = 3x + 1\)

Sumar \(k\) → sube · Restar \(k\) → baja
📌 Ejemplo 2 — Función Cuadrática
\(f(x) = x^2 + 3\)

3 unidades arriba: \(u(x) = x^2 + 3 + 3 = x^2 + 6\)
6 unidades abajo: \(d(x) = x^2 + 3 - 6 = x^2 - 3\)

El vértice se desplaza verticalmente.
TEORÍA

Traslación Horizontal — Ejemplos

04 / 20

Ejemplos de traslación horizontal para distintos tipos de funciones.

📌 Ejemplo 1 — Función Lineal
\(f(x) = 2x\)

2 unidades derecha: \(r(x) = 2(x - 2) = 2x - 4\)
4 unidades izquierda: \(l(x) = 2(x + 4) = 2x + 8\)

Restar \(h\) → derecha · Sumar \(h\) → izquierda
📌 Ejemplo 2 — Función Cuadrática
\(f(x) = 3x^2 - 2\)

1 unidad derecha: \(r(x) = 3(x - 1)^2 - 2\)
5 unidades izquierda: \(l(x) = 3(x + 5)^2 - 2\)

El vértice se desplaza horizontalmente.
⚠️
¡Cuidado! En traslación horizontal, el signo es contrario a lo que se podría pensar: \(f(x-2)\) se mueve a la derecha.

Vértice de Funciones Cuadráticas

05 / 20

En la forma canónica \(f(x) = (x - h)^2 + k\), el vértice es \((h, k)\).

📌 Identificar el vértice
\(f(x) = (x + 3)^2 - 2\) → \(V(-3, -2)\)
\(g(x) = (x - 5)^2 - 1\) → \(V(5, -1)\)
\(h(x) = x^2 + 3\) → \(V(0, 3)\)
\(k(x) = (x - 8)^2\) → \(V(8, 0)\)
📌 Traslación y vértice
\(f(x) = x^2\) (vértice en \((0,0)\))

Trasladar 5 unidades a la derecha:
\(g(x) = (x - 5)^2\) → \(V(5, 0)\)

Trasladar 8 unidades hacia arriba:
\(g(x) = x^2 + 8\) → \(V(0, 8)\)

Traslaciones en Otras Funciones (M2)

06 / 20

Las traslaciones también se aplican a funciones potencia, logarítmicas, exponenciales y trigonométricas.

📌 Función Logarítmica
\(f(x) = \log x\)

5 unidades derecha y 8 arriba:
\(g(x) = \log(x - 5) + 8\)
📌 Función Exponencial
\(f(x) = 2^x\)

3 unidades izquierda y 4 abajo:
\(g(x) = 2^{x+3} - 4\)
📌 Consideraciones especiales
  • Para funciones trigonométricas, la traslación horizontal cambia la fase.
  • Para funciones potencia, la traslación sigue la misma regla: \(f(x-h)+k\).

Análisis de Funciones Lineales

07 / 20

Aplicaciones de funciones lineales en problemas de proporcionalidad directa.

📌 Cálculo de pendiente
\(P_1(2, 1)\) y \(P_2(6, 3)\)
\(m = \frac{3 - 1}{6 - 2} = \frac{2}{4} = 0.5\)
📌 Coeficiente de posición
Usando \(P_1(2, 1)\) y \(m = 0.5\):
\(n = 1 - 0.5(2) = 0\)
Ecuación: \(f(x) = 0.5x\)
📌 Identificación de gráficas
La gráfica de \(f(x) = 2x - 1\) es una recta con pendiente \(2\) y corte en \((0, -1)\).

Análisis de Funciones Cuadráticas

08 / 20

Análisis completo de funciones cuadráticas: raíces, vértice, coeficientes y gráfica.

📌 Elementos clave
  • Raíces: \(x_1, x_2\) (intersecciones con \(x\))
  • Vértice: \(V(h, k)\) (máximo o mínimo)
  • Coeficientes: \(a, b, c\) de \(ax^2 + bx + c\)
  • Concavidad: \(a > 0\) → arriba, \(a < 0\) → abajo
📌 Ejemplo
\(f(x) = x^2 - 4x + 3\)

\(a = 1, b = -4, c = 3\)
Raíces: \(x = 1\) y \(x = 3\)
Vértice: \(V(2, -1)\)
Concavidad: \(a > 0\) → arriba

Aplicación: Precio de Frutas

09 / 20
📝 Problema
Una persona compró 30 kg de naranjas en $24.000 y 30 kg de manzanas en $27.000.

Si la cantidad de kilogramos de frutas y su precio se modelan por medio de una relación lineal, ¿cuál de las siguientes expresiones representa el precio total al comprar \(x\) kg de cada fruta?
📌 Análisis
  • Precio naranjas: \(\frac{24.000}{30} = 800\) por kg
  • Precio manzanas: \(\frac{27.000}{30} = 900\) por kg
  • Precio total: \(800x + 900x = 1700x\)
La expresión correcta es:
\(\left(\frac{24.000}{30} + \frac{27.000}{30}\right)x = 1.700x\)
📌 Respuesta
La alternativa correcta es:

\(\frac{24.000}{30}x + \frac{27.000}{30}x\)

Respuesta: \(1.700x\)

Aplicación: Chirridos del Grillo

10 / 20
📝 Problema
La cantidad \(n\) de chirridos por minuto que emite un grillo se relaciona con la temperatura ambiental mediante la ecuación: \[ T = 10 + \frac{n - 5}{12} \] donde \(T\) es la temperatura ambiental en °C.

¿Cuántos chirridos por minuto emitirá un grillo si la temperatura ambiental es de \(19°C\)?
📌 Solución paso a paso
\(19 = 10 + \frac{n - 5}{12}\)

\(9 = \frac{n - 5}{12}\)

\(108 = n - 5\)

\(n = 113\)
📌 Respuesta
El grillo emitirá 113 chirridos por minuto.

Alternativa correcta: c) 113

Aplicación: Función Cuadrática

11 / 20
📝 Problema
Considera la función cuadrática \(f\), cuyo dominio es el conjunto de los números reales, definida por: \[ f(x) = ax^2 + bx + c \] Si los coeficientes \(b\) y \(c\) son cero, ¿cuál de los siguientes gráficos podría corresponder a la gráfica asociada a \(f\)?
📌 Análisis
  • \(b = 0\) → el vértice está en \(x = 0\)
  • \(c = 0\) → la gráfica corta al eje \(y\) en \((0,0)\)
  • La función es \(f(x) = ax^2\)
  • Es una parábola con vértice en el origen
📌 Respuesta
La única gráfica que coincide es:

Una parábola con vértice en \((0,0)\)

(La alternativa correspondiente a esta gráfica)

Aplicación: Trayectoria de un Proyectil

12 / 20
📝 Problema
Un proyectil sigue una trayectoria parabólica, de modo que la función que describe su altura \(H(t)\) (en metros), dependiendo del tiempo transcurrido \(t\) (en segundos) es: \[ H(t) = -(t - 2)^2 + 35 \] ¿Cuál es la altura máxima que alcanza este proyectil?
📌 Análisis
  • La función está en forma canónica
  • \(H(t) = -(t - 2)^2 + 35\)
  • Vértice: \((2, 35)\)
  • \(a = -1 < 0\) → el vértice es un máximo
  • Altura máxima: \(35\) metros
📌 Respuesta
La altura máxima es 35 metros.

Alternativa correcta: d) 35

Resolución de Problemas

14 / 20
📝 Problema 1 — Costo de viaje
El costo de un viaje en taxi está dado por \(C(d) = 500 + 200d\), donde \(d\) es la distancia en km.

¿Cuánto cuesta un viaje de 8 km?

Solución: \(C(8) = 500 + 200(8) = 2.100\)
Respuesta: $2.100
📝 Problema 2 — Lanzamiento
La altura de un objeto está dada por \(h(t) = -5t^2 + 30t\). ¿Cuál es la altura máxima?

Solución: \(t = -\frac{30}{2(-5)} = 3\), \(h(3) = -45 + 90 = 45\)
Respuesta: 45 m
📝 Problema 3 — Temperatura
La temperatura en una ciudad está dada por \(T(h) = 20 - 2h\), donde \(h\) es la altura en km.

¿Cuál es la temperatura a 4 km de altura?

Solución: \(T(4) = 20 - 2(4) = 12\)°C
Respuesta: 12°C

Resumen: Análisis de Funciones

20 / 20
📌 Contenidos clave
  • Traslación vertical: \(f(x) + k\) → sube (\(k > 0\)), baja (\(k < 0\))
  • Traslación horizontal: \(f(x - h)\) → derecha (\(h > 0\)), izquierda (\(h < 0\))
  • Traslación general: \(g(x) = f(x - h) + k\)
  • Vértice de función cuadrática: \(V(h, k)\) en forma canónica
  • Funciones lineales: Pendiente \(m\), corte \(n\)
  • Funciones cuadráticas: \(a, b, c\), vértice, raíces
  • Máximos y mínimos: Vértice determina el punto extremo
🎯 Estrategias para el examen
  • Identifica el tipo de función y sus parámetros.
  • Para traslaciones: \(f(x-h)+k\) → \(h\) derecha, \(k\) arriba.
  • Recuerda: \(f(x+h)\) → \(h\) unidades a la izquierda.
  • El vértice de \(a(x-h)^2 + k\) es \((h, k)\).
  • Para máximos/mínimos: encuentra el vértice.
  • En problemas verbales: traduce a lenguaje algebraico.
  • Verifica siempre con un punto conocido de la función.
