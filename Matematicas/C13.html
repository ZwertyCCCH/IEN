Definición de Función Cuadrática

02 / 20

Una función cuadrática es una función polinómica de grado 2. Se define como:

\[ f(x) = ax^2 + bx + c \quad \text{con } a \neq 0 \] a, b, c ∈ ℝ · a: coeficiente cuadrático · b: coeficiente lineal · c: término independiente
📌 Ejemplos
\(f(x) = x^2 - 4x + 3\)
\(f(x) = -2x^2 + 5x - 1\)
\(f(x) = 3x^2 - 12\)
\(f(x) = -x^2 + 6x\)
📌 Gráfica
  • Su gráfica es una parábola
  • Puede abrir hacia arriba (\(a > 0\))
  • Puede abrir hacia abajo (\(a < 0\))
  • Tiene un vértice (máximo o mínimo)
ELEMENTOS

Concavidad

03 / 20

La concavidad de una parábola está determinada por el signo del coeficiente \(a\).

📌 Concavidad hacia arriba
\(a > 0\) → Cóncava hacia arriba
  • La parábola se abre hacia arriba
  • El vértice es un mínimo
  • La función decrece y luego crece
\(f(x) = x^2 - 4x + 3\) → \(a = 1 > 0\)
📌 Concavidad hacia abajo
\(a < 0\) → Cóncava hacia abajo
  • La parábola se abre hacia abajo
  • El vértice es un máximo
  • La función crece y luego decrece
\(f(x) = -x^2 + 4x - 3\) → \(a = -1 < 0\)
ELEMENTOS

Eje de Simetría

04 / 20

El eje de simetría es una recta vertical que divide la parábola en dos partes simétricas.

\[ x = -\frac{b}{2a} \] También se puede calcular como \(h = \frac{x_1 + x_2}{2}\)
📌 Ejemplo
\(f(x) = x^2 - 4x + 3\)

\(a = 1, b = -4\)
\(x = -\frac{-4}{2(1)} = \frac{4}{2} = 2\)

Eje de simetría: \(x = 2\)
📌 Propiedades
  • La parábola es simétrica respecto al eje.
  • El vértice está sobre el eje de simetría.
  • Si \(x_1\) y \(x_2\) son raíces, \(h = \frac{x_1 + x_2}{2}\)
ELEMENTOS

Vértice

05 / 20

El vértice es el punto más alto o más bajo de la parábola. Se calcula como:

\[ V = \left(-\frac{b}{2a}, \frac{4ac - b^2}{4a}\right) = (h, f(h)) \]
📌 Ejemplo 1
\(f(x) = x^2 - 4x + 3\)

\(h = -\frac{-4}{2(1)} = 2\)
\(k = f(2) = 4 - 8 + 3 = -1\)

Vértice: \(V(2, -1)\)
📌 Ejemplo 2
\(f(x) = -x^2 + 6x - 5\)

\(a = -1, b = 6\)
\(h = -\frac{6}{2(-1)} = 3\)
\(k = f(3) = -9 + 18 - 5 = 4\)

Vértice: \(V(3, 4)\) (máximo)
📌
Interpretación: Si \(a > 0\), el vértice es un mínimo. Si \(a < 0\), el vértice es un máximo.
ELEMENTOS

Intersecciones con el Eje X

06 / 20

Las intersecciones con el eje \(x\) son los puntos donde \(f(x) = 0\), es decir, las raíces de la ecuación cuadrática.

\[ f(x) = 0 \iff ax^2 + bx + c = 0 \] Se resuelve con la fórmula general o factorización
✅ 2 intersecciones
\(\Delta > 0\)
\(f(x) = x^2 - 5x + 6\)
Raíces: \(x = 2, 3\)
✅ 1 intersección
\(\Delta = 0\)
\(f(x) = x^2 - 4x + 4\)
Raíz: \(x = 2\) (doble)
❌ 0 intersecciones
\(\Delta < 0\)
\(f(x) = x^2 + 2x + 5\)
No tiene raíces reales
ELEMENTOS

Intersección con el Eje Y

07 / 20

La intersección con el eje \(y\) es el punto donde la parábola cruza el eje vertical, es decir, cuando \(x = 0\).

\[ f(0) = c \quad \Rightarrow \quad (0, c) \]
📌 Ejemplo
\(f(x) = x^2 - 4x + 3\)

\(f(0) = 0^2 - 4(0) + 3 = 3\)

Intersección: \((0, 3)\)
📌 Características
  • La parábola siempre corta al eje \(y\).
  • El punto de corte es \((0, c)\).
  • \(c\) es el término independiente.
\(f(x) = -2x^2 + 5x - 3\) → \((0, -3)\)
ELEMENTOS

Dilatación y Contracción

08 / 20

El coeficiente \(a\) también determina la amplitud de la parábola.

📌 Contracción
\(|a| > 1\) → Parábola más cerrada
\(f(x) = 2x^2\)
Es más cerrada que \(x^2\)
📌 Dilatación
\(0 < |a| < 1\) → Parábola más abierta
\(f(x) = \frac{1}{2}x^2\)
Es más abierta que \(x^2\)
📌
Importante: \(|a|\) grande → parábola angosta · \(|a|\) pequeño → parábola ancha.

Forma Canónica

09 / 20

La forma canónica de una función cuadrática permite identificar directamente el vértice de la parábola.

\[ f(x) = a(x - h)^2 + k \] (h, k) es el vértice de la parábola
📌 Ejemplo 1
\(f(x) = (x - 3)^2 + 2\)

\(a = 1 > 0\) → mínimo
\(V = (3, 2)\)
📌 Ejemplo 2
\(f(x) = -2(x + 1)^2 - 4\)

\(a = -2 < 0\) → máximo
\(V = (-1, -4)\)
📌 Transformación
\(f(x) = x^2 - 4x + 3\)
\(f(x) = (x^2 - 4x + 4) - 4 + 3\)
\(f(x) = (x - 2)^2 - 1\)
Vértice: \(V(2, -1)\)

Ejemplos tipo examen

10 / 20

Ejercicios similares a las preguntas PAES sobre funciones cuadráticas.

Ejemplo 1 — Vértice
Encuentra el vértice de \(f(x) = x^2 - 6x + 8\).
Solución: \(h = -\frac{-6}{2(1)} = 3\), \(k = f(3) = 9 - 18 + 8 = -1\)
Respuesta: \(V(3, -1)\)
Ejemplo 2 — Concavidad
¿Cuál es la concavidad de \(f(x) = -3x^2 + 2x - 5\)?
Solución: \(a = -3 < 0\) → cóncava hacia abajo
Respuesta: Hacia abajo (máximo)
Ejemplo 3 — Intersecciones
Encuentra las intersecciones con el eje \(x\) de \(f(x) = x^2 - 4x + 3\).
Solución: \(x^2 - 4x + 3 = 0 \rightarrow (x-1)(x-3) = 0 \rightarrow x = 1, 3\)
Respuesta: \((1, 0)\) y \((3, 0)\)

Resolución de Problemas

11 / 20
📝 Problema 1 — Altura máxima
Un proyectil sigue la trayectoria \(h(t) = -5t^2 + 40t\), donde \(t\) está en segundos.

a) ¿Cuál es la altura máxima?
b) ¿En qué tiempo la alcanza?

Solución: \(h(t) = -5(t^2 - 8t) = -5[(t-4)^2 - 16] = -5(t-4)^2 + 80\)
Respuesta: Altura máxima 80 m a los 4 segundos.
📝 Problema 2 — Área máxima
Se quiere construir un rectángulo con 40 m de perímetro. ¿Cuál es el área máxima?

Solución: \(2(x + y) = 40 \rightarrow y = 20 - x\)
\(A(x) = x(20 - x) = -x^2 + 20x\)
\(h = -\frac{20}{2(-1)} = 10\), \(A(10) = 100\)
Respuesta: Área máxima 100 m².
📝 Problema 3 — Tiro parabólico
La altura de un objeto está dada por \(h(t) = -4t^2 + 16t + 20\).

¿En qué tiempo el objeto toca el suelo?

Solución: \(-4t^2 + 16t + 20 = 0 \rightarrow t^2 - 4t - 5 = 0\)
\((t - 5)(t + 1) = 0 \rightarrow t = 5\)
Respuesta: A los 5 segundos.

Funciones Cuadráticas en Contextos Navales

16 / 20

Las funciones cuadráticas modelan trayectorias parabólicas, optimización y diseño en contextos navales.

⚓ Trayectoria de un proyectil
Un proyectil sigue la trayectoria \(h(x) = -0.02x^2 + 0.8x + 10\), donde \(x\) es la distancia horizontal en metros.

a) ¿Cuál es la altura máxima?
b) ¿A qué distancia horizontal ocurre?

Solución: \(h = -\frac{0.8}{2(-0.02)} = 20\), \(k = -0.02(400) + 16 + 10 = 18\)
Respuesta: Altura máxima 18 m a 20 m de distancia.
📐 Diseño de un arco
Un arco tiene forma parabólica con ecuación \(f(x) = -0.5x^2 + 4x\).

¿Cuál es la altura máxima del arco?

Solución: \(h = -\frac{4}{2(-0.5)} = 4\), \(k = -0.5(16) + 16 = 8\)
Respuesta: Altura máxima 8 metros.
📊 Optimización de recursos
El costo de producción de un buque está dado por \(C(x) = 2x^2 - 40x + 250\).

¿Cuántas unidades minimizan el costo?

Solución: \(x = -\frac{-40}{2(2)} = \frac{40}{4} = 10\)
Respuesta: 10 unidades.

Más Aplicaciones Prácticas

17 / 20
📈 Lanzamiento de un objeto
La altura de un objeto está dada por \(h(t) = -5t^2 + 30t + 10\).

a) ¿Cuál es la altura máxima?
b) ¿En qué tiempo la alcanza?

Solución: \(t = -\frac{30}{2(-5)} = 3\), \(h(3) = -45 + 90 + 10 = 55\)
Respuesta: 55 m a los 3 segundos.
📐 Área de un terreno
Se tiene 60 m de cerca para cercar un terreno rectangular. ¿Cuál es el área máxima que se puede cercar?

Solución: \(x(30 - x) = -x^2 + 30x\) → \(h = 15\), \(A(15) = 225\)
Respuesta: Área máxima 225 m².
🧪 Concentración de una solución
La concentración de una solución está dada por \(C(t) = -t^2 + 4t + 12\).

¿Cuál es la concentración máxima?

Solución: \(t = -\frac{4}{2(-1)} = 2\), \(C(2) = -4 + 8 + 12 = 16\)
Respuesta: Concentración máxima 16.

Ejercicios Avanzados

19 / 20
Ejercicio 4 — Parámetro a
Encuentra el valor de \(a\) para que \(f(x) = ax^2 + 4x + 1\) tenga vértice en \(x = -2\).
Solución: \(-\frac{4}{2a} = -2 \rightarrow \frac{4}{2a} = 2 \rightarrow 4 = 4a \rightarrow a = 1\)
Respuesta: \(a = 1\)
Ejercicio 5 — Raíces y vértice
Una función cuadrática tiene raíces \(x = 1\) y \(x = 5\), y vértice en \((3, -4)\). Encuentra la función.
Solución: \(f(x) = a(x-1)(x-5)\), \(f(3) = a(2)(-2) = -4a = -4 \rightarrow a = 1\)
Respuesta: \(f(x) = x^2 - 6x + 5\)
Ejercicio 6 — Problema de optimización
Un buque tiene un costo de operación \(C(x) = 0.5x^2 - 10x + 200\). ¿Cuál es el costo mínimo?
Solución: \(x = \frac{10}{1} = 10\), \(C(10) = 0.5(100) - 100 + 200 = 150\)
Respuesta: Costo mínimo 150.

Resumen: Funciones Cuadráticas

20 / 20
📌 Contenidos clave
  • Forma general: \(f(x) = ax^2 + bx + c\)
  • Concavidad: \(a > 0\) → arriba, \(a < 0\) → abajo
  • Eje de simetría: \(x = -\frac{b}{2a}\)
  • Vértice: \(V\left(-\frac{b}{2a}, \frac{4ac - b^2}{4a}\right)\)
  • Intersección con \(x\): Resolver \(ax^2 + bx + c = 0\)
  • Intersección con \(y\): \((0, c)\)
  • Forma canónica: \(f(x) = a(x - h)^2 + k\)
  • Dilatación/Contracción: \(|a| > 1\) → cerrada, \(|a| < 1\) → abierta
🎯 Estrategias para el examen
  • Identifica \(a, b, c\) en la función.
  • Para el vértice: usa \(x = -\frac{b}{2a}\).
  • Para intersecciones con \(x\): resuelve la ecuación cuadrática.
  • Para intersección con \(y\): evalúa en \(x = 0\).
  • La forma canónica da el vértice directamente.
  • En problemas de optimización: busca el vértice.
  • Recuerda: \((x - h)^2 + k\) → vértice \((h, k)\).
