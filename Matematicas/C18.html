Estadística Descriptiva

02 / 20

La estadística descriptiva emplea métodos y técnicas para la recolección, ordenamiento, resumen, análisis, interpretación y comunicación de datos.

📌 Conceptos clave
  • Población: Conjunto total de individuos o elementos de interés.
  • Muestra: Subconjunto de la población que se estudia.
  • Encuesta: Forma de recabar información.
  • Variable: Característica de la población que se estudia.
📌 Tipos de variables
  • Cuantitativas: Se pueden medir o contar.
    Ej: Estatura, masa, tiempo.
  • Cualitativas: Características no numéricas.
    Ej: Color, nacionalidad, rango militar.
📌
Importante: Una muestra representativa permite hacer inferencias sobre toda la población.

Tablas de Frecuencias

03 / 20

Una tabla de frecuencias organiza los datos y resume la información de manera clara y ordenada.

📌 Elementos de una tabla
  • Dato (x): Valor de la variable.
  • Frecuencia (f): Número de veces que se repite.
  • Frecuencia acumulada (F): Suma de frecuencias.
  • Frecuencia relativa (fr): \(f / n\) (fracción).
  • Frecuencia porcentual (%): \(fr \cdot 100\).

📌 Elementos

  • Límite inferior y superior: Extremos del intervalo.
  • Amplitud: Diferencia entre límites.
  • Marca de clase (MC): Promedio de los límites.

Media Aritmética (Promedio)

07 / 20

La media aritmética o promedio es el cociente entre la suma de todos los datos y la cantidad de datos.

\[ \bar{x} = \frac{\sum_{i=1}^{n} x_i}{n} = \frac{x_1 + x_2 + x_3 + \cdots + x_n}{n} \]
📌 Ejemplo (datos no agrupados)
Datos: \(2, 3, 4, 4, 6, 6, 8, 9\)
\(\bar{x} = \frac{2+3+4+4+6+6+8+9}{8} = \frac{42}{8} = 5.25\)
📌 Ejemplo (datos agrupados)
\(\bar{x} = \frac{\sum MC_i \cdot f_i}{n}\)
\(= \frac{50\cdot20 + 150\cdot30 + 250\cdot50}{100} = \frac{17500}{100} = 175\)
📌
Importante: La media es sensible a valores extremos (outliers).

Mediana

08 / 20

La mediana es el valor que divide al conjunto de datos ordenados en dos partes iguales.

📌 n impar
Datos: \(2, 3, 4, 4, 6, 6, 8, 9, 10\)
\(n = 9\) → posición central: \(5\)
Mediana: \(6\)
📌 n par
Datos: \(2, 3, 4, 4, 6, 6, 8, 9\)
\(n = 8\) → posiciones centrales: \(4\) y \(5\)
Mediana: \(\frac{4+6}{2} = 5\)
📌 Intervalo que contiene a la mediana
Se calcula \(\frac{n}{2}\) y se busca el intervalo en la frecuencia acumulada.
Si \(\frac{n}{2}\) es decimal, se redondea al entero mayor.
MEDIDAS DE TENDENCIA CENTRAL

Moda

09 / 20

La moda es el valor que más se repite en un conjunto de datos.

✅ Unimodal
\(2, 3, 4, 4, 6, 6, 8, 9\)
Moda = 6 (se repite 2 veces)
✅ Polimodal
\(2, 3, 4, 4, 6, 6, 8, 8, 9\)
Modas = 4, 6, 8
❌ Amodal
\(2, 2, 4, 4, 9, 9\)
Sin moda (todas se repiten igual)
📌
Para datos agrupados: Se habla de intervalo modal (el de mayor frecuencia).

Resolución de Problemas

11 / 20
📝 Problema 1 — Promedio de notas
Un estudiante tiene las siguientes notas: 4.5, 5.0, 3.5, 6.0, 5.5. ¿Cuál es su promedio?

Solución: \(\frac{4.5+5.0+3.5+6.0+5.5}{5} = \frac{24.5}{5} = 4.9\)
Respuesta: 4.9
📝 Problema 2 — Mediana de edades
En un curso, las edades son: 18, 19, 20, 21, 22, 23, 24. ¿Cuál es la mediana?

Solución: \(n=7\) → posición 4 → 21 años
Respuesta: 21 años
📝 Problema 3 — Moda en una encuesta
En una encuesta sobre color favorito: rojo (12), azul (15), verde (10), azul (8). ¿Cuál es la moda?

Solución: Azul con 15 (se repite más)
Respuesta: Azul

Estadística en Contextos Navales

16 / 20

La estadística es fundamental en la navegación, logística y planificación naval.

⚓ Promedio de velocidad
Un buque registra las siguientes velocidades (nudos): 22, 25, 28, 24, 26. ¿Cuál es la velocidad promedio?

Solución: \(\bar{x} = \frac{22+25+28+24+26}{5} = \frac{125}{5} = 25\) nudos
Respuesta: 25 nudos
📊 Consumo de combustible
Los consumos diarios (L) son: 120, 130, 125, 140, 135, 130. ¿Cuál es la mediana?

Solución: Ordenados: 120, 125, 130, 130, 135, 140
Mediana: \(\frac{130+130}{2} = 130\) L
Respuesta: 130 L
🧭 Moda en temperaturas
Las temperaturas diarias (°C) son: 18, 20, 22, 20, 19, 21, 20. ¿Cuál es la moda?

Solución: 20 se repite 3 veces
Respuesta: 20°C

Resumen: Análisis Gráfico y Medidas de Tendencia Central

20 / 20
📌 Contenidos clave
  • Población: Conjunto total de individuos.
  • Muestra: Subconjunto representativo.
  • Frecuencia (f): Número de veces que se repite un dato.
  • Frecuencia acumulada (F): Suma de frecuencias.
  • Frecuencia relativa (fr): \(f/n\)
  • Marca de clase (MC): Promedio de los límites.
  • Media: \(\bar{x} = \frac{\sum x_i}{n}\)
  • Mediana: Valor central (n impar) o promedio de centrales (n par).
  • Moda: Valor que más se repite.
  • Histograma: Barras juntas para intervalos.
  • Gráfico de barras: Barras separadas para discretos/cualitativos.
🎯 Estrategias para el examen
  • Identifica el tipo de variable (cualitativa/cuantitativa).
  • Para media: suma todos los datos y divide por \(n\).
  • Para mediana: ordena los datos y encuentra el centro.
  • Para moda: cuenta la frecuencia de cada valor.
  • En tablas con intervalos: usa marca de clase.
  • Interpreta gráficos para extraer conclusiones.
  • Recuerda: la media es sensible a outliers.
