TEORÍA

Definición de Función

02 / 20

Una función es una regla que asigna a cada elemento \(x\) de un conjunto \(A\) uno y solo un elemento \(y\) de un conjunto \(B\).

\[ f: A \to B \quad \text{con} \quad y = f(x) \] x: variable independiente · y: variable dependiente
📌 Ejemplo
\(f(x) = 2x + 3\)

\(f(1) = 2(1) + 3 = 5\)
\(f(2) = 2(2) + 3 = 7\)
\(f(3) = 2(3) + 3 = 9\)
📌 Analogía
Materia prima → Función → Producto final

\(x\) (materia prima) → \(f\) (transformación) → \(f(x)\) (producto)

Ejemplo: \(x = 2\) → \(f(x) = 2x + 3\) → \(f(2) = 7\)
💡
Importante: Una función no puede tener dos imágenes diferentes para el mismo \(x\). Es la regla de la unicidad.

Dominio y Recorrido

03 / 20

El dominio es el conjunto de valores que puede tomar la variable \(x\). El recorrido es el conjunto de valores que toma \(f(x)\).

📌 Definiciones
  • Dominio (Dom f): Conjunto de pre-imágenes (valores de \(x\)).
  • Recorrido (Rec f): Conjunto de imágenes (valores de \(f(x)\)).
  • Co-dominio: Conjunto de llegada \(B\).
📌 Ejemplo
\(f(x) = x^2\)

Dominio: \(\mathbb{R}\)
Recorrido: \([0, \infty[\)

\(f(2) = 4\), \(f(-2) = 4\)
Ambas pre-imágenes tienen la misma imagen.
📌
Importante: El dominio son los valores que entran a la función. El recorrido son los valores que salen.

Evaluación de Funciones

04 / 20

Para evaluar una función, reemplazamos la variable \(x\) por el valor o expresión indicado.

📌 Imagen de un valor
\(f(x) = 2x + 5\)

\(f(4) = 2 \cdot 4 + 5 = 13\)
\(f(-3) = 2(-3) + 5 = -1\)
\(f(0) = 2(0) + 5 = 5\)

Notación: \(f(4) = 13\) se lee "f de 4 es 13".

📌 Pre-imagen de un valor
\(f(x) = 2x + 5\)

Buscar la pre-imagen de 19:
\(19 = 2x + 5\)
\(2x = 14 \rightarrow x = 7\)

Notación: \(f^{-1}(19) = 7\)
📌
Notación: \(f^{-1}(y)\) se lee "pre-imagen de \(y\)" (no confundir con función inversa).

Representación Gráfica

05 / 20

Una función se puede representar en el plano cartesiano mediante los puntos \((x, f(x))\).

📌 Puntos de una función
\(f(x) = 2x + 1\)

\(x=0 \rightarrow f(0)=1\) → \((0, 1)\)
\(x=1 \rightarrow f(1)=3\) → \((1, 3)\)
\(x=2 \rightarrow f(2)=5\) → \((2, 5)\)
\(x=3 \rightarrow f(3)=7\) → \((3, 7)\)
📌 Gráfica
📌
Importante: La gráfica de una función no puede tener dos puntos con el mismo \(x\) pero diferente \(y\).
TEORÍA

Funciones en el Plano Cartesiano

06 / 20

En el plano cartesiano, el eje \(x\) representa la variable independiente y el eje \(y\) representa la variable dependiente \(f(x)\).

📌 Elementos del plano
  • Eje \(x\) (abscisas): Valores de entrada.
  • Eje \(y\) (ordenadas): Valores de salida.
  • Origen: Punto \((0, 0)\).
  • Punto: \((x, f(x))\).
📌 Ejemplo visual
📌
Recuerda: La gráfica de una función es el conjunto de todos los puntos \((x, f(x))\) para \(x\) en el dominio.

Valorización de Funciones

07 / 20

Ejercicios para practicar la evaluación de funciones y la búsqueda de pre-imágenes.

📌 Ejemplo 1
\(f(x) = 3x - 2\)

\(f(2) = 3(2) - 2 = 4\)
\(f(-1) = 3(-1) - 2 = -5\)
\(f(0) = 3(0) - 2 = -2\)
📌 Ejemplo 2
\(f(x) = x^2 + 2x\)

\(f(3) = 9 + 6 = 15\)
\(f(-2) = 4 - 4 = 0\)
\(f(0) = 0 + 0 = 0\)
📌 Ejemplo 3 — Buscar pre-imagen
\(f(x) = 2x + 1\)
Buscar \(f^{-1}(7)\):
\(7 = 2x + 1 \rightarrow 2x = 6 \rightarrow x = 3\)
Respuesta: \(f^{-1}(7) = 3\)

Función Constante

08 / 20

La función constante asigna el mismo valor a cualquier \(x\). Es de la forma:

\[ f(x) = n \quad \text{con } n \in \mathbb{R} \]
📌 Tabla de valores

📌 Características
  • Dominio: \(\mathbb{R}\)
  • Recorrido: \(\{n\}\)
  • Gráfica: Recta horizontal
  • Pendiente: \(m = 0\)
  • El valor \(n\) es constante para todo \(x\).

Función Lineal

09 / 20

Una función lineal es de la forma \(f(x) = mx\) con \(m \neq 0\). Su gráfica es una recta que pasa por el origen.

\[ f(x) = mx \quad \text{con } m \in \mathbb{R}, \; m \neq 0 \] m: pendiente de la recta
📌 Características
  • Dominio: \(\mathbb{R}\)
  • Recorrido: \(\mathbb{R}\)
  • Pasa por: \((0,0)\)
  • Si \(m > 0\) → creciente
  • Si \(m < 0\) → decreciente
📌 Ejemplos
\(f(x) = 2x\) → \(m = 2\) (creciente)
\(f(x) = -3x\) → \(m = -3\) (decreciente)
\(f(x) = \frac{1}{2}x\) → \(m = 0.5\) (creciente)

Función Identidad

10 / 20

La función identidad asigna a cada \(x\) su mismo valor. Es un caso especial de función lineal con \(m = 1\).

\[ f(x) = x \]
📌 Tabla de valores

📌 Características
  • Dominio: \(\mathbb{R}\)
  • Recorrido: \(\mathbb{R}\)
  • Pendiente: \(m = 1\)
  • Pasa por: \((0,0)\), \((1,1)\), \((-1,-1)\)
  • Es una función biyectiva

Función Afín

11 / 20

Una función afín es de la forma \(f(x) = mx + n\) con \(m \neq 0\). Su gráfica es una recta que no necesariamente pasa por el origen.

\[ f(x) = mx + n \quad \text{con } m, n \in \mathbb{R}, \; m \neq 0 \] m: pendiente · n: coeficiente de posición
📌 Características
  • Dominio: \(\mathbb{R}\)
  • Recorrido: \(\mathbb{R}\)
  • Corte con \(y\): \((0, n)\)
  • Si \(m > 0\) → creciente
  • Si \(m < 0\) → decreciente
📌 Ejemplos
\(f(x) = 2x + 3\) → \(m=2, n=3\)
\(f(x) = -x + 5\) → \(m=-1, n=5\)
\(f(x) = \frac{1}{2}x - 4\) → \(m=0.5, n=-4\)

Ejemplos tipo examen

12 / 20

Ejercicios similares a las preguntas PAES sobre funciones.

Ejemplo 1 — Evaluación
Si \(f(x) = 3x - 2\), ¿cuál es el valor de \(f(4)\)?
Solución: \(f(4) = 3(4) - 2 = 12 - 2 = 10\)
Respuesta: \(f(4) = 10\)
Ejemplo 2 — Pre-imagen
Si \(f(x) = 2x + 1\), encuentra \(f^{-1}(7)\).
Solución: \(7 = 2x + 1 \rightarrow 2x = 6 \rightarrow x = 3\)
Respuesta: \(f^{-1}(7) = 3\)
Ejemplo 3 — Dominio y recorrido
¿Cuál es el dominio y recorrido de \(f(x) = x^2 + 1\)?
Solución: Dominio = \(\mathbb{R}\), Recorrido = \([1, \infty[\)
Respuesta: Dom = \(\mathbb{R}\), Rec = \([1, \infty[\)

Resolución de Problemas

13 / 20
📝 Problema 1 — Costo de viaje
El costo de un viaje en taxi está dado por \(C(d) = 500 + 200d\), donde \(d\) es la distancia en km. ¿Cuánto cuesta un viaje de 8 km?

Solución: \(C(8) = 500 + 200(8) = 500 + 1600 = 2.100\)
Respuesta: El viaje cuesta $2.100.
📝 Problema 2 — Distancia
Un auto recorre una distancia dada por \(d(t) = 60t\), donde \(t\) es el tiempo en horas. ¿Qué distancia recorre en 2.5 horas?

Solución: \(d(2.5) = 60(2.5) = 150\) km
Respuesta: Recorre 150 km.
📝 Problema 3 — Temperatura
La temperatura en °C está dada por \(T(h) = 20 - 2h\), donde \(h\) es la altura en km. ¿Cuál es la temperatura a 4 km de altura?

Solución: \(T(4) = 20 - 2(4) = 20 - 8 = 12\)°C
Respuesta: La temperatura es 12°C.

Preguntas estilo examen

Pregunta 1 — PAES M1
Si \(f(x) = 2x^2 - 3x + 1\), ¿cuál es el valor de \(f(2)\)?

a) 3   b) 4   c) 5   d) 6
Solución: \(f(2) = 2(4) - 3(2) + 1 = 8 - 6 + 1 = 3\)
Respuesta: a) 3
Pregunta 2 — PAES M1
¿Cuál es la pre-imagen de 5 en la función \(f(x) = 2x + 1\)?

a) 1   b) 2   c) 3   d) 4
Solución: \(5 = 2x + 1 \rightarrow 2x = 4 \rightarrow x = 2\)
Respuesta: b) 2
Pregunta 3 — PAES M1
La gráfica de una función \(f\) pasa por el punto \((2, 5)\) y su pendiente es \(3\). ¿Cuál es la ecuación de la función afín?

a) \(f(x) = 3x - 1\)   b) \(f(x) = 3x + 5\)   c) \(f(x) = 3x - 5\)   d) \(f(x) = 5x + 3\)
Solución: \(f(x) = 3x + n\) → \(5 = 3(2) + n \rightarrow n = -1\) → \(f(x) = 3x - 1\)
Respuesta: a) \(f(x) = 3x - 1\)

Funciones en Contextos Navales

18 / 20

Las funciones permiten modelar relaciones en navegación, logística y operaciones navales.

⚓ Velocidad de un buque
La velocidad de un buque está dada por \(v(t) = 25 - 2t\), donde \(t\) es el tiempo en horas. ¿Cuál es la velocidad a las 3 horas?

Solución: \(v(3) = 25 - 2(3) = 25 - 6 = 19\) nudos
Respuesta: La velocidad es 19 nudos.
⛽ Consumo de combustible
El consumo de combustible de un motor está dado por \(C(h) = 150 + 80h\), donde \(h\) es el número de horas. ¿Cuánto combustible se consume en 4 horas?

Solución: \(C(4) = 150 + 80(4) = 150 + 320 = 470\) L
Respuesta: Se consumen 470 litros.
📊 Profundidad de un submarino
La profundidad de un submarino está dada por \(P(t) = -50t - 100\), donde \(t\) es el tiempo en minutos. ¿Cuál es la profundidad a los 2 minutos?

Solución: \(P(2) = -50(2) - 100 = -100 - 100 = -200\) m
Respuesta: La profundidad es 200 m bajo el nivel del mar.

Más Aplicaciones Prácticas

19 / 20
📐 Área de un cuadrado
El área de un cuadrado está dada por \(A(l) = l^2\), donde \(l\) es el lado. ¿Cuál es el área si el lado mide 7 cm?

Solución: \(A(7) = 7^2 = 49\) cm²
Respuesta: El área es 49 cm².
📊 Conversión de temperatura
La conversión de °C a °F está dada por \(F(C) = \frac{9}{5}C + 32\). ¿Cuántos °F son 20°C?

Solución: \(F(20) = \frac{9}{5}(20) + 32 = 36 + 32 = 68\)°F
Respuesta: 20°C son 68°F.
🧭 Distancia de frenado
La distancia de frenado de un vehículo está dada por \(d(v) = \frac{v^2}{20}\), donde \(v\) es la velocidad en m/s. ¿Cuál es la distancia de frenado a 10 m/s?

Solución: \(d(10) = \frac{100}{20} = 5\) m
Respuesta: La distancia de frenado es 5 m.

Resumen: Funciones

20 / 20
📌 Contenidos clave
  • Función: Regla que asigna a cada \(x\) un único \(f(x)\)
  • Dominio: Valores que puede tomar \(x\)
  • Recorrido: Valores que puede tomar \(f(x)\)
  • Evaluación: \(f(a)\) = valor de la función en \(x=a\)
  • Pre-imagen: \(f^{-1}(y)\) = valor de \(x\) tal que \(f(x)=y\)
  • Función constante: \(f(x)=n\)
  • Función lineal: \(f(x)=mx\)
  • Función identidad: \(f(x)=x\)
  • Función afín: \(f(x)=mx+n\)
🎯 Estrategias para el examen
  • Identifica el tipo de función.
  • Para evaluar: sustituye el valor en la variable.
  • Para pre-imagen: iguala la función al valor y despeja.
  • Para dominio: identifica restricciones (raíces, denominadores).
  • Recuerda: \(\sqrt{x}\) → \(x \geq 0\), \(\frac{1}{x}\) → \(x \neq 0\).
  • Para funciones lineales y afines: \(m\) es la pendiente, \(n\) es el corte con \(y\).
