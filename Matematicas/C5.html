Monomios y Polinomios

03 / 20

Un término algebraico combina números y letras. Según la cantidad de términos, las expresiones se clasifican en monomios, binomios, trinomios o polinomios.

Monomio (1 término)
\(x\)
\(3x\)
\(2x^2y\)
\(15x^3y^4\)
Binomio (2 términos)
\(x + y\)
\(3x + 5y^2\)
\(x^2 - 4\)
\(7x^3 - 2x^5\)
Trinomio (3 términos)
\(2x + 3y - 4z\)
\(x^2 + 3x - 7\)
\(x^2 + 2xy + y^2\)
Polinomio (2+ términos)
\(x + 3y\)
\(x^5 - 3x^4 + 2x^3 + 11x^2 - x + 9\)
\(x^2 - y^2 + z^3 + 1\)
📌 Grado de un término

El grado es la suma de los exponentes de todas las variables en el factor literal.

\(5x^2\) → grado 2  ·  \(x^3y\) → grado 4  ·  \(x^2y^3z\) → grado 6
OPERACIONES

Términos Semejantes

04 / 20

Los términos semejantes tienen el mismo factor literal (mismas letras con los mismos exponentes). Se pueden sumar o restar.

📌 Ejemplo 1
\(12ab + 4c + 6ab\)
\(= 18ab + 4c\)

Nota: \(12ab\) y \(6ab\) son semejantes. \(4c\) no lo es.

📌 Ejemplo 2
\(9m^2 + 7n - 4m^2 - n\)
\(= 5m^2 + 6n\)

Nota: \(9m^2\) y \(-4m^2\) son semejantes. \(7n\) y \(-n\) son semejantes.

📌 Eliminación de paréntesis
Regla 1: Si el paréntesis está precedido por \(+\), se elimina sin cambios.
Regla 2: Si el paréntesis está precedido por \(-\), se cambia el signo de todos los términos.

\(-(-(-(-x))) + (-(-(-(-x)))) = x + x = 2x\)
OPERACIONES

Multiplicación de Polinomios

05 / 20
📌 Monomio × Monomio

Multiplicar coeficientes y factores literales.

\((3a^2b^3) \cdot (2a^5c)\)
\(= 6a^7b^3c\)
📌 Monomio × Polinomio

Aplicar propiedad distributiva.

\(-3m(2+m-4n)\)
\(= -6m - 3m^2 + 12mn\)
📌 Polinomio × Polinomio

Multiplicar cada término del primero por cada término del segundo.

\((x+2)(y-3)\)
\(= xy - 3x + 2y - 6\)
📌
Recuerda: La propiedad distributiva dice que \(a(b+c) = ab + ac\). Se aplica siempre en multiplicación de polinomios.
PRODUCTOS NOTABLES

Cuadrado de Binomio

06 / 20
\[ (a \pm b)^2 = a^2 \pm 2ab + b^2 \]
📌 Con suma
\((x+1)^2 = x^2 + 2x + 1\)
\((x+5)^2 = x^2 + 10x + 25\)
\((x+10)^2 = x^2 + 20x + 100\)
📌 Con resta
\((x-1)^2 = x^2 - 2x + 1\)
\((x-5)^2 = x^2 - 10x + 25\)
\((x-10)^2 = x^2 - 20x + 100\)
🧠
Truco: "El cuadrado del primero, más/menos el doble producto del primero por el segundo, más el cuadrado del segundo".
PRODUCTOS NOTABLES

Suma por Diferencia

07 / 20
\[ (a + b)(a - b) = a^2 - b^2 \]
📌 Ejemplos
\((x+1)(x-1) = x^2 - 1\)
\((x+3)(x-3) = x^2 - 9\)
\((x+7)(x-7) = x^2 - 49\)
📌 Con coeficientes
\((2x+3)(2x-3) = 4x^2 - 9\)
\((5x+2)(5x-2) = 25x^2 - 4\)
\((x^2+4)(x^2-4) = x^4 - 16\)
📌
Importante: El resultado siempre es la diferencia de cuadrados \(a^2 - b^2\).
PRODUCTOS NOTABLES

Binomios con Término Común

08 / 20
\[ (x + a)(x + b) = x^2 + (a + b)x + ab \]
📌 Ambos positivos
\((x+2)(x+3) = x^2 + 5x + 6\)
\((x+4)(x+7) = x^2 + 11x + 28\)
\((x+6)(x+10) = x^2 + 16x + 60\)
📌 Uno negativo
\((x+3)(x-2) = x^2 + x - 6\)
\((x-7)(x+3) = x^2 - 4x - 21\)
\((x-10)(x+17) = x^2 + 7x - 170\)
📌
Regla: El término de grado 1 es la suma de los términos independientes. El término de grado 0 es su producto.
PRODUCTOS NOTABLES

Cubo de Binomio

09 / 20
\[ (a \pm b)^3 = a^3 \pm 3a^2b + 3ab^2 \pm b^3 \]
📌 Con suma
\((x+1)^3 = x^3 + 3x^2 + 3x + 1\)
\((x+2)^3 = x^3 + 6x^2 + 12x + 8\)
\((x+3)^3 = x^3 + 9x^2 + 27x + 27\)
📌 Con resta
\((x-1)^3 = x^3 - 3x^2 + 3x - 1\)
\((x-2)^3 = x^3 - 6x^2 + 12x - 8\)
\((x-3)^3 = x^3 - 9x^2 + 27x - 27\)
🧮
Patrón: Coeficientes 1, 3, 3, 1. Los signos alternan en la resta: \(+ - + -\).
FACTORIZACIÓN

Factor Común

10 / 20

La factorización es el proceso inverso al producto. El factor común consiste en extraer el factor que se repite en todos los términos.

\[ ax + bx = x(a + b) \]
📌 Ejemplos
\(x^2 + 2x = x(x+2)\)
\(x^3 - 5x^2 = x^2(x-5)\)
\(2x^3 - 7x = x(2x^2-7)\)
\(12x + 3 = 3(4x+1)\)
📌 Factor común compuesto
\(x+y-2x^2-2xy\)
\(= (x+y) - 2x(x+y)\)
\(= (x+y)(1-2x)\)

\(3x+5+6xy+10y\)
\(= (3x+5) + 2y(3x+5)\)
\(= (3x+5)(1+2y)\)
FACTORIZACIÓN

Diferencia de Cuadrados

11 / 20
\[ a^2 - b^2 = (a + b)(a - b) \]
📌 Ejemplos
\(x^2 - y^2 = (x+y)(x-y)\)
\(x^2 - 4 = (x+2)(x-2)\)
\(x^2 - 49 = (x+7)(x-7)\)
📌 Con coeficientes
\(4x^2 - 9y^2 = (2x+3y)(2x-3y)\)
\(x^4 - 16 = (x^2+4)(x^2-4)\)
\(25x^2 - 1 = (5x+1)(5x-1)\)
📌
Identifica: Debes tener una resta de dos cuadrados perfectos para aplicar esta factorización.
FACTORIZACIÓN

Trinomio Cuadrado Perfecto

12 / 20
\[ a^2 + 2ab + b^2 = (a + b)^2 \] \[ a^2 - 2ab + b^2 = (a - b)^2 \]
📌 Ejemplos
\(x^2 + 2x + 1 = (x+1)^2\)
\(x^2 + 14x + 49 = (x+7)^2\)
\(x^2 + 20x + 100 = (x+10)^2\)
📌 Con resta
\(x^2 - 4x + 4 = (x-2)^2\)
\(x^2 - 16x + 64 = (x-8)^2\)
\(x^2 - 10x + 25 = (x-5)^2\)
🔍
Verificación: El término de grado 1 debe ser el doble producto de las raíces cuadradas del primer y último término.
FACTORIZACIÓN

Trinomio Cuadrado Imperfecto

13 / 20
\[ x^2 + (a + b)x + ab = (x + a)(x + b) \]
📌 Ambos positivos
\(x^2 + 5x + 6 = (x+2)(x+3)\)
\(x^2 + 4x + 3 = (x+3)(x+1)\)
\(x^2 + 7x - 30 = (x+10)(x-3)\)
📌 Uno negativo
\(x^2 - 2x - 15 = (x-5)(x+2)\)
\(x^2 - 7x + 12 = (x-4)(x-3)\)
\(x^2 + 4x - 117 = (x+13)(x-9)\)
🧩
Estrategia: Busca dos números que sumen el coeficiente de \(x\) y multipliquen al término independiente.

Resolución de Problemas

15 / 20
📝 Problema 1 — Área de un cuadrado
El lado de un cuadrado mide \((x+3)\) cm. ¿Cuál es su área?

Solución: \(A = (x+3)^2 = x^2 + 6x + 9\)
Respuesta: El área es \(x^2 + 6x + 9\) cm².
📝 Problema 2 — Producto de binomios
El largo de un rectángulo mide \((x+5)\) y el ancho \((x-2)\). ¿Cuál es su área?

Solución: \(A = (x+5)(x-2) = x^2 + 3x - 10\)
Respuesta: El área es \(x^2 + 3x - 10\).
📝 Problema 3 — Diferencia de cuadrados
Un terreno cuadrado de lado \(x\) tiene un área de \(144\) m². Encuentra el valor de \(x\).

Solución: \(x^2 = 144 \rightarrow x = 12\)
Respuesta: \(x = 12\) metros.

Pregunta 1 — PAES M1
Si \(x=3\), ¿cuál es el valor de \(x^2 + 2x - 1\)?

a) 10   b) 12   c) 14   d) 16
Solución: \(3^2 + 2\cdot3 - 1 = 9 + 6 - 1 = 14\)
Respuesta: c) 14
Pregunta 2 — PAES M1
¿Cuál es el resultado de \((x+2)^2\)?

a) \(x^2+4\)   b) \(x^2+2x+4\)   c) \(x^2+4x+4\)   d) \(x^2+4x+2\)
Solución: \((x+2)^2 = x^2 + 2\cdot x\cdot2 + 2^2 = x^2 + 4x + 4\)
Respuesta: c) \(x^2+4x+4\)
Pregunta 3 — PAES M1
¿Cuál es la factorización de \(x^2 - 9\)?

a) \((x-9)(x+1)\)   b) \((x+3)(x-3)\)   c) \((x+9)(x-1)\)   d) \((x-3)^2\)
Solución: \(x^2 - 9 = x^2 - 3^2 = (x+3)(x-3)\)
Respuesta: b) \((x+3)(x-3)\)

RESUMEN — CLASE 05

Resumen: Expresiones Algebraicas

20 / 20
📌 Contenidos clave
  • Variable: Letra que representa un valor desconocido.
  • Monomio: 1 término (ej: \(3x^2y\))
  • Binomio: 2 términos (ej: \(x+y\))
  • Trinomio: 3 términos (ej: \(x^2+2xy+y^2\))
  • Términos semejantes: Mismo factor literal.
  • Cuadrado de binomio: \((a \pm b)^2 = a^2 \pm 2ab + b^2\)
  • Suma por diferencia: \((a+b)(a-b)=a^2-b^2\)
  • Binomios con término común: \((x+a)(x+b)=x^2+(a+b)x+ab\)
  • Cubo de binomio: \((a\pm b)^3 = a^3 \pm 3a^2b + 3ab^2 \pm b^3\)
  • Factorización: Factor común, diferencia de cuadrados, trinomio cuadrado perfecto, trinomio cuadrado imperfecto.
🎯 Estrategias para el examen
  • Identifica términos semejantes antes de simplificar.
  • Para productos notables: memoriza las fórmulas y practica.
  • Para factorización: identifica el tipo de expresión.
  • Verifica tus factorizaciones multiplicando.
  • En problemas con valor numérico: sustituye y calcula.
  • Recuerda: \((a+b)^2 \neq a^2 + b^2\) (falta el término \(2ab\)).
